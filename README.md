## Instructions

With this app a user can track books that they are currently reading, would like to read, or have read already. 

* For each book the user can select which category (shelf) they would like the book to go into by selecting the shelf in the drop down menu located at the bottom of each book. 
* The user can search for additional titles by selecting the "+" icon in the bottom right corner of the app.
* To search for a title the can simply type in the title. 
* Once the user has searched for a title they would like to add to the collection, they can add the title to a shelf in the same way they moved titles on the main screen, by selecting the shelf they would like the title to be added to by selecting the shelf in the drop down menu located at the bottom of each book.
* The user can navigate back to the main page by clicking the arrow located in the search field.

## Installation
To install this application, the user will need to:

* Download or clone this repository
* In terminal, "cd" into the downloaded or cloned repository
* Install all project dependencies with `npm install`
* Start the server with `npm start`
